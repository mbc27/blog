# 博客系统功能优化

## Core Features

- 未登录用户只能访问首页、看博客、友链、联系我页面

- 登录后可使用全部功能

- 将'健客'修改为'博客'

- 头像功能完善，点击可进入个人中心或退出

## Design

博客系统采用Vue2 + Element UI构建，后端使用Spring Boot + MySQL，实现了基本的博客功能，包括文章展示、发布、评论等功能，以及用户权限管理。
